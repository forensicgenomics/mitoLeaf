Report an Issue

If you encounter a bug, error, think something is missing / not displayed properly or have a suggestion for improving the website, please provide the following information when reporting the issue:

  • A clear and concise description of the problem
  • Steps to reproduce the issue
  • Your browser and version (if applicable)
  • Any screenshots or error messages (Possibly including your browser console messages)

You can report issues directly on our GitHub repository. Click the button below to create a new issue. If you're unfamiliar with GitHub issues, you can also check this guide for more information.
